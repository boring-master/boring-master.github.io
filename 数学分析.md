
# 不定积分
## 基本积分公式
- $\int \sec^2x \,dx = \tan x + C$ 
- $\int \csc^2x \,dx = -\cot x + C$
- $\int \sec x \cdot \tan x \,dx = \sec x + C$ 
- $\int \csc x \cdot \cot x \,dx = -\csc x + C$
- $\int \frac{dx}{\sqrt{1-x^2}} = \arcsin x + C = -\arccos x + C_1$
- $\int \frac{dx}{1+x^2} = \arctan x + C = -\operatorname{arccot} x + C_1$
## 积分方法
### 换元积分法
#### **第一换元积分法(凑微分法)**:
- 若$\int f(x)\,dx = F(x) + C$存在，则
$$\int f(\varphi(t))\varphi'(t) \,dt = F(\varphi(t)) + C$$
#### **第二换元积分法(代入换元法)**:
- $t = \varphi^{-1}(x)$，若$\int f(\varphi(t))\varphi'(t) \,dt = G(t) + C$存在，则
$$\int f(x) \,dx = G(\varphi^{-1}(x)) + C$$
### 分部积分法
- 若$\int u'(x)v(x) \,dx$存在，则
$$\int u(x)v'(x) \,dx = u(x)v(x) - \int u'(x)v(x) \,dx$$
# 定积分
设$[a,b]$上有$n-1$个点，依次为$a=x_0<x_1<\dots<x_n=b$，记$\Delta_i=[x_{i-1},x_i]，i=1,2,\dots,n.$记分割$T = \{x_0,x_1,\dots,x_n\}$或$\{\Delta_1,\Delta_2,\dots,\Delta_n\}.$小区间的长度为$\Delta x_i = x_i-x_{i-1}$，并记分割$T$的模为$\|T\| = \max_{1 \leq i \leq n}\{\Delta x_i\}$
## 牛顿-莱布尼茨公式
若$F'(x) = f(x)$，则$\int_{a}^{b} f(x)\,dx = F(b) - F(a)$
## 可积条件
$M_i = \sup_{x \in \Delta_i}f(x)$，$m_i = \inf_{x \in \Delta_i}f(x)$，$i = 1,2,\dots,n$.   
$f$关于分割$T$的上和与下和(或称达布上和与达布下和，统称达布和)分别为  
$S(T) = \sum_{i = 1}^{n}M_i\Delta x_i$，$s(T) = \sum_{i = 1}^{n}m_i \Delta x_i$.  

### 可积的必要条件
- 若函数$f$在$[a,b]$上可积，则$f$在$[a,b]$上必定有界
### 可积的充要条件
- (可积准则)函数$f$在$[a,b]$上可积的充要条件是：任给$\varepsilon > 0$，总存在相应的一个分割$T$，使得$S(T) - s(T) < \varepsilon$

$f$在$\Delta_i$上的振幅为$w_i = M_i - m_i$，必要时记为$w_i^f$，由于$S(T) - s(T) = \sum_{i = 1}^{n}w_i\Delta x_i$(或记为$\sum_T w_i\Delta x_i$)，所以又表述为如下形式
- 函数$f$在$[a,b]$上可积的充要条件是：任给$\varepsilon > 0$，总存在相应的某一分割$T$，使得$\sum_{T}w_i\Delta x_i < \varepsilon$
### 可积函数类
- 若$f$为$[a,b]$上的连续函数，则$f$在$[a,b]$上可积
- 若$f$是区间$[a,b]$上只有有限个间断点的有界函数，则$f$在$[a,b]$上可积
- 若$f$是$[a,b]$上的单调函数，则$f$在$[a,b]$上可积
## 积分中值定理
### 积分第一中值定理
- 若$f$在$[a,b]$上连续，则至少存在一点$\xi \in [a,b]$，使得
$$\int_a^b f(x) \,dx = f(\xi)(b-a)$$
### 推广的积分第一中值定理
- 若$f$与$g$都在$[a,b]$上连续，且$g(x)$在$[a,b]$上不变号，则至少存在一点$\xi \in [a,b]$，使得
$$\int_a^b f(x)g(x) \,dx = f(\xi)\int_a^b g(x) \,dx$$
## 微积分学基本定理
### 变限积分与原函数的存在性
变上限的积分为$\phi(x) = \int_a^x f(t) \,dt$，$x \in [a,b]$  
变下限的积分为$\psi(x) = \int_x^b f(t) \,dt$，$x \in [a,b]$

$\phi$与$\psi$统称为变限积分
- 若$f$在$[a,b]$上可积，则$\phi$在$[a,b]$上连续
- (原函数存在定理)若$f$在$[a,b]$上连续，则$\phi$在$[a,b]$上处处可导，且
$$\phi'(x) = \frac{d}{dx}\int_a^x f(t) \,dt = f(x)，x \in [a,b]$$
- (积分第二中值定理)设函数$f$在$[a,b]$上可积.
    1. 若$g$在$[a,b]$上减，且$g(x) \geqslant 0$，则存在$\xi \in [a,b]$，使得
    $$\int_a^b f(x)g(x) \,dx = g(a)\int_a^{\xi} f(x) \,dx；$$
    2. 若$g$在$[a,b]$上增，且$g(x) \geqslant 0$，则存在$\eta \in [a,b]$，使
    $$\int_a^b f(x)g(x) \,dx = g(b)\int_{\eta}^b f(x) \,dx.$$
- 设函数$f$在$[a,b]$上可积，若$g$为单调函数，则存在$\xi \in [a,b]$，使得
$$\int_a^b f(x)g(x) \,dx = g(a)\int_a^{\xi} f(x) \,dx + g(b)\int_{\xi}^b f(x) \,dx.$$
### 换元积分法与分部积分法
#### 定积分换元积分法
- 若函数$f$在$[a,b]$上连续，$\varphi'$在$[a,b]$上可积，且满足$\varphi(\alpha) = a$，$\varphi(\beta) = b$，$\varphi([\alpha,\beta]) \subset [a,b]$，则有：
$$\int_a^b f(x) \,dx = \int_{\alpha}^{\beta} f(\varphi(t))\varphi'(t) \,dt$$
#### 定积分分部积分法
- 若$u(x)$，$v(x)$为$[a,b]$上的可微函数，且$u'(x)$和$v'(x)$都在$[a,b]$上可积，则有：
$$\int_a^b u(x)v'(x) \,dx = u(x)v(x)|_a^b - \int_a^b u'(x)v(x) \,dx$$
## 可积性理论补叙
### 上和与下和的性质
$f$在$[a,b]$上的上、下确界为$M$与$m$  
$f$在$[a,b]$上的上积分为$S = \inf_T S(T)$，$f$在$[a,b]$上的下积分为$s = \sup_T s(T)$
- 对同一个分割$T$，相对于任何点集$\{\xi_i\}$而言，上和是所有积分和的上确界，下和是所有积分和的下确界，即
$$S(T) = \sup_{\{\xi_i\}} \sum_{i = 1}^n f(\xi_i) \Delta x_i，s(T) = \inf_{\{\xi_i\}} \sum_{i = 1}^n f(\xi) \Delta x_i$$
- 设$T'$为分割$T$添加$p$个新分点后所得到的分割，则有上和不增，下和不减，即
$$S(T) \geq S(T') \geq S(T) - (M - m)p\|T\|,$$
$$s(T) \leq s(T') \leq s(T) + (M - m)p\|T\|.$$
- 若$T'$与$T''$为任意两个分割，$T = T' + T''$表示吧$T'$与$T''$的所有分点合并而得的分割(重复的点只取一次)，则
$$S(T) \leq S(T')，s(T) \geq s(T'),$$
$$S(T) \leq S(T'')，s(T) \geq s(T'').$$
- 对任意的两个分割$T'$与$T''$，总有
$$s(T') \leq S(T'')$$
- $$m(b-a) \leq s \leq S \leq M(b-a)$$
- (达布定理)上、下积分也是上和与下和在$\|T\|\rightarrow0$时的极限，即
$$\lim_{\|T\|\rightarrow0}S(T) = S，\lim_{\|T\|\rightarrow0}s(T) = s.$$
### 可积的充要条件
- (可积的第一充要条件)函数$f$在$[a,b]$上可积的充要条件是：$f$在$[a,b]$上的上积分与下积分相等，即
$$S=s$$
- (可积的第二充要条件)函数$f$在$[a,b]$上可积的充要条件是：任给$\varepsilon > 0$，总存在某一分割$T$，使得
$$S(T) - s(T) < \varepsilon，即\sum_{i=1}^n w_i\Delta x_i < \varepsilon$$
- (可积的第三充要条件)函数$f$在$[a,b]$上可积的充要条件是：任给正数$\varepsilon$，$\eta$，总存在某一分割$T$，使得属于$T$的所有小区间中，对应于振幅$w_{k'} \geq \varepsilon$的那些小区间$\Delta_{k'}$的总长
$$\sum_{k'} \Delta x_{k'} < \eta$$
